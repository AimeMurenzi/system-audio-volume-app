# System Audio Volume App

This app controls system audio volume, allowing you to increase
or decrease audio volume of a selected audio output line via a web app. 
## Tech Stack
 
- **Java**
- **Spring**
- **Spring Boot** 
- **Spring Web** 
- **Thymeleaf** 
- **HTML5**
- **CSS3**
- **Javascript**
- **OpenFX**
